# Project 2 - Iris Data Classification Using KNN

## Description

This project implements a Machine Learning classification model using the K-Nearest Neighbors (KNN) algorithm on the Iris dataset. The model classifies iris flowers into three species: Setosa, Versicolor, and Virginica.

## Features

* Loads the Iris dataset from Scikit-Learn
* Splits data into training and testing sets
* Applies feature scaling using StandardScaler
* Trains a K-Nearest Neighbors (KNN) classifier
* Generates predictions on test data
* Evaluates model performance using:
* F1 Score
* Classification Report
* Confusion Matrix
* Visualizes the confusion matrix using Seaborn and Matplotlib

## Technologies Used

* Python
* Scikit-Learn
* NumPy
* Pandas
* Seaborn
* Matplotlib

## Dataset Information

* Dataset: Iris Dataset
* Total Samples: 150
* Features: 4
* Classes:
    Setosa,
    Versicolor,
    Virginica.

## Results

The model achieved:

F1 Score: 1.0, 
Accuracy: 100%

The confusion matrix showed that all test samples were classified correctly.
